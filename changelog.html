Changelog

See what's new in SnapText. All notable changes are documented here.

v2.0.3 April 6, 2026 Latest
Added
  • New variable types: {{uppercase}}, {{lowercase}}, {{random:}}, {{calc:}}, {{select:}}, {{date+/-N}}
  • Auto-suggest feature shows matching macros as you type
  • Custom trigger character support (change from ; to any character)
  • Keyboard shortcut customization for quick search
  • Abbreviation mode - expand macros without trigger prefix
  • Dark mode for website with system preference detection
  • PWA support - install SnapText dashboard as an app
  • Browser detection for platform-specific install instructions
  • Mobile-friendly hamburger menu navigation
  • Live expansion counter on homepage
  • New pages: Use Cases, Documentation, Changelog
Improved
  • Replaced deprecated document.execCommand with modern Input Events API
  • Better mobile experience with responsive design improvements
  • Enhanced settings UI in dashboard
v2.0.2 April 4, 2026
Added
  • Web dashboard for browser-based macro management
  • Full landing page with feature showcase
  • Demo animation showing text expansion in action
v2.0.1 April 3, 2026
Added
  • Version stamp in popup header
Fixed
  • Resilient popup loading with inline defaults
  • Promise-based message handlers for better reliability
v2.0.0 April 2, 2026 Major
Added
  • Firebase cloud sync - back up and sync macros across devices
  • Macro sharing with 8-character share codes
  • Explore library for discovering public macro packs
  • Folder management - create, rename, delete folders
  • CSV and JSON import/export
  • Quick search overlay (Ctrl+Shift+Space)
  • Full dashboard for managing macros and settings
  • Dynamic variables: {{date}}, {{time}}, {{input:}}, {{cursor}}, {{clipboard}}
  • Context-based macro suggestions
Improved
  • Complete UI redesign with modern blue/white theme
  • Better text insertion with undo support
  • Improved contenteditable field compatibility
v1.0.0 March 2026
Added
  • Initial release
  • Basic text expansion with ; prefix
  • Local storage for macros
  • Simple popup interface for managing snippets
  • Works in standard text inputs and textareas
