Documentation

Everything you need to know about using SnapText effectively.

Installation

SnapText is installed manually as a Chrome extension. Follow these steps:

  1. Download the extension from the homepage or direct link
  2. Unzip the downloaded file to a folder on your computer
  3. Open Chrome and go to chrome://extensions
  4. Enable Developer mode (toggle in top-right corner)
  5. Click "Load unpacked" and select the unzipped folder
Tip: SnapText works on any Chromium-based browser including Chrome, Edge, Brave, Arc, and Opera.

Your First Macro

Let's create a simple email signature macro:

  1. Click the SnapText icon in your browser toolbar
  2. Click "Add Macro"
  3. Enter a trigger: sig
  4. Enter your expansion text: Best regards,\nYour Name
  5. Click "Save"

Now type ;sig followed by Space in any text field to insert your signature!

Trigger Characters

By default, macros are triggered by typing a semicolon (;) followed by the trigger word. You can customize this in Settings:

  • Common alternatives: /, :, !, @
  • The trigger must be followed by a space, tab, or enter to expand

User Input Variables

Prompt the user for input when the macro expands:

VariableDescription
{{input:Label}}Shows a text input with the label
{{select:A,B,C}}Shows a dropdown with options A, B, C

Example:

Hi {{input:Name}},

Thank you for contacting us about {{select:billing,technical support,general inquiry}}.

Best regards

Auto-Suggest

When enabled, SnapText shows matching macros as you type. After typing your trigger character, suggestions appear automatically.

  • Use ↑ / ↓ to navigate suggestions
  • Press Tab or Enter to select
  • Press Esc to dismiss

Toggle auto-suggest in Settings.

Abbreviation Mode

Abbreviation mode lets specific macros expand without the trigger prefix. Enable it per-macro in the macro editor.

Example: Set "btw" as an abbreviation. Typing "btw " will expand to "by the way" without needing ";btw".

Note: Use abbreviation mode sparingly to avoid accidental expansions.

Folder Management

Organize macros into folders:

  1. Open the Dashboard
  2. Click "+ New Folder" in the sidebar
  3. Name your folder (e.g., "Work", "Personal", "Code")
  4. Drag macros to folders or assign when editing

Tags

Tags provide another way to organize and filter macros:

  1. Open a macro for editing
  2. Add tags in the Tags field (comma-separated)
  3. Click the tag filter dropdown in the toolbar to filter by tag

Examples: "email", "code", "urgent", "template", "personal"

Tip: Use tags for cross-cutting categories that don't fit neatly into folders, like "high-priority" or "needs-review".

Snippet Templates

SnapText includes pre-built template packs to get you started quickly:

  • Email Essentials - Professional email responses (follow-up, thank you, introduction)
  • Developer Toolkit - Code snippets and templates (console.log, function, try-catch)
  • Support Responses - Customer support templates (acknowledgment, resolution)
  • Meeting Templates - Agendas and follow-up emails
  • Fun & Casual - Emojis and casual responses (shrug, lenny)
  • Sales Outreach - Sales and prospecting templates

Access templates from the Templates tab in Share & Explore.

Multi-Cursor Support

Place multiple cursor positions in your macros. After expansion, press Tab to jump between positions.

Dear {{cursor:1}},

Thank you for your {{cursor:2}}.

Best regards,
{{cursor:3}}

When this macro expands, press Tab to jump from position 1 to 2 to 3.

VariableDescription
{{cursor}}Single cursor (legacy syntax)
{{cursor:1}}First cursor position
{{cursor:2}}Second cursor position
{{cursor:N}}Nth cursor position

Snippet Chaining

Link macros together so that expanding one automatically triggers another:

  1. Edit a macro in the dashboard
  2. Find the "Chain to" dropdown
  3. Select another macro to trigger after this one expands

Example: Create a greeting macro that chains to a signature macro for complete email templates.

Note: Chains have a maximum depth of 5 to prevent infinite loops.

Usage Analytics

Track your macro usage with the built-in analytics dashboard:

  • Usage Chart - See expansions over time (7-day view)
  • Top Macros - Discover your most-used snippets
  • Activity Heatmap - See which days and hours you use macros most
  • Total Stats - Track total expansions and characters saved

Access Analytics from the sidebar in the dashboard.

Setting Up Cloud Sync

Cloud sync backs up your macros and syncs them across devices:

  1. Open the Dashboard
  2. Click "Sign In"
  3. Sign in with your Google account
  4. Your macros will automatically sync
Privacy: Your macros are stored securely in Firebase. Only you can access your data.

Import & Export

Export your macros for backup or transfer:

  • JSON export: Full data including folders and settings
  • CSV export: Simple format for spreadsheet editing

Import from the Dashboard by clicking the import button and selecting your file.

Import from Other Tools

SnapText can import macros from other text expansion tools:

TextExpander Import

Import from TextExpander CSV exports:

  1. In TextExpander, export your snippets as CSV
  2. In SnapText, go to Import and select "TextExpander CSV"
  3. Select your exported file
  4. Preview the imported macros and confirm

Mapped fields: abbreviation → trigger, snippet → body, label → folder

Espanso Import

Import from Espanso YAML configuration files:

  1. Locate your Espanso config file (usually default.yml)
  2. In SnapText, go to Import and select "Espanso YAML"
  3. Select your config file
  4. Preview and confirm the import

Mapped fields: trigger → trigger, replace → body

Note: Espanso variables are automatically converted to SnapText syntax where possible.

Troubleshooting: Macros Not Expanding

If macros aren't expanding, try these steps:

  1. Check the trigger: Make sure you're using the correct prefix (default: ;)
  2. Add a space: Type space, tab, or enter after the trigger to expand
  3. Refresh the page: The content script may need to reload
  4. Check the field type: Some websites use custom editors that may not be compatible
  5. Disable conflicting extensions: Other text expanders may interfere

Troubleshooting: Sync Issues

If sync isn't working:

  1. Check your connection: Ensure you're online
  2. Re-authenticate: Sign out and sign back in
  3. Force sync: Click the sync button in the dashboard
  4. Check for conflicts: If the same macro was edited on multiple devices, you may need to resolve conflicts
